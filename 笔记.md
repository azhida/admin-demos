# 笔记

使用 pnpm 实现 单仓库 多项目

## 安装 pnpm

```sh
npm i -g pnpm
pnpm -v
# 8.6.6
```

## 构建根目录项目 & 添加 workspace 配置

```sh
pnpm init
touch pnpm-workspace.yaml
```

/pnpm-workspace.yaml
```yaml
packages:
  - 'packages/*'
```

## 安装 ts
```sh
pnpm add  -D -w typescript
```
`-D` ：作为开发依赖安装
`-w` ： `monorepo` 环境默认会认为应该将依赖安装到具体的 `package` 中。使用 -w 参数，告诉 pnpm 将依赖安装到 workspace-root，也就是项目的根目录。

## 构建子项目
```sh
mkdir packages
cd packages
```

- 添加 启动指令

/package.json

```json
{

```

```sh
pnpm run dev:demo-admin-elementplus
```
